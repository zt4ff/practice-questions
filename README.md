# Practice Question

- Open Pull Requests for the questions you want to make available via https://www.olukayode.tech/goodies/checkpoint

- Indicate to make the questions public or private to you alone via a secret code.

- They are several content type:
    - lesson (plain text, video (Youtube Iframe))
    - coding questions
    - mcq questions
    - interview questions (answering an AI)

- see `sample.ts` for an example
